# MoneyForKids - Financial Education Website

A modern, mobile-responsive website redesign for financial education platform targeting children and parents. This project transforms the original moneyforkids.in into a contemporary, user-friendly experience.

## 🚀 Features

### Design & User Experience
- **Modern Design**: Clean, professional layout with gradient backgrounds and smooth animations
- **Mobile-First**: Fully responsive design that works perfectly on all devices
- **Interactive Elements**: Hover effects, smooth transitions, and engaging animations
- **Accessibility**: Keyboard navigation, focus states, and screen reader friendly

### Technical Features
- **Responsive Navigation**: Collapsible mobile menu with smooth animations
- **Smooth Scrolling**: Seamless navigation between sections
- **Form Validation**: Client-side validation for contact forms
- **Performance Optimized**: Lazy loading, debounced scroll events, and efficient animations
- **Touch Gestures**: Swipe support for mobile navigation

### Content Sections
1. **Hero Section**: Compelling headline with call-to-action buttons
2. **Features**: Key benefits of the platform
3. **About**: Company information with animated statistics
4. **Courses**: Age-appropriate learning programs
5. **Resources**: Free downloadable materials
6. **Testimonials**: Parent reviews and feedback
7. **Contact**: Contact form and company information
8. **Footer**: Links, social media, and newsletter signup

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript (ES6+)**: Interactive functionality
- **Font Awesome**: Icons and visual elements
- **Google Fonts**: Inter font family for typography
- **Unsplash**: High-quality placeholder images

## 📱 Mobile Responsiveness

The website is fully responsive with breakpoints at:
- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: 480px - 767px
- **Small Mobile**: Below 480px

### Mobile Features
- Collapsible hamburger menu
- Touch gesture support (swipe to open/close menu)
- Optimized typography and spacing
- Touch-friendly buttons and interactive elements

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Local web server (optional, for development)

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. For development, use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

### File Structure
```
├── index.html          # Main HTML file
├── styles.css          # CSS styles and responsive design
├── script.js           # JavaScript functionality
└── README.md           # Project documentation
```

## 🔧 Customization

### Colors
Update the CSS custom properties in `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --text-color: #333;
    --text-light: #666;
    --bg-light: #f8f9fa;
}
```

### Content
- Update text content in `index.html`
- Replace placeholder images with your own
- Modify contact information and social media links

### Styling
- Adjust spacing, typography, and colors in `styles.css`
- Modify animations and transitions
- Customize mobile breakpoints
